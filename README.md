# PixelShrink

> Batch image compression — shrink images without sacrificing quality.

## Features

- Compress PNG, JPG, WebP, and AVIF images in batches
- Adjust quality slider from 0–100 for fine-grained control
- Choose from presets: Maximum, High, Medium, Low
- Strip EXIF metadata for smaller file sizes
- Resize images to specific dimensions before compression
- Compare original vs. compressed size and quality side by side
- Download compressed images individually or as a ZIP
- Process hundreds of files simultaneously
- See real-time file size savings before downloading

## Why PixelShrink?

All compression happens locally using browser-native APIs and WebAssembly codecs. Zero images are uploaded to any server. PixelShrink is free, private, and works offline — making it the ideal tool for optimizing images for the web, email, or storage without compromising your privacy.
